We’re proud to present to you our massive update to Bold! After being selected in the Envato’s 2011 Birthday Bundle and turning up as one of ThemeForest’s favorite minimal style CV/Resume template, it’s time for a substantial update to BOLD.

Bold v.2 – The Bestselling CV/Resume HTML template just got Bolder!

Bold: Included bonuses

Printable (300dpi, CMYK) PSD files for your resume and assorted (all 15 color themes) business cards in portrait and landscape mode!

Bold: New features

Responsive resume with a minimal layout, sortable portfolio with responsive image and video lightbox, improved typography, 15 color themes, 11 background effects and full size photo background option.

Bold: Resources

  • Font used: Open Sans
  • Icons used: Font Awesome
  • Preview video: Satisfaction – Superstars
  • Scripts used: jQuery, jQuery form plugin, jQuery form validation plugin, Backstretch, MixItUp, Magnific Popup

Updates

v2.0 – August 21, 2013 – Massive update: responsive layout, added color themes, improved typography, sortable portfolio and many more improvements and bonuses.

v1.4.1 – May 25, 2013 – Small CSS fix for mobile display. Overwriting your own schemes/mobile.css with the one included in the download package will apply the fix.

v1.4 – January 25, 2013 – Updated files to add migration support to jQuery 1.9.0. This fixes an issue that mainly manifested with lightbox- type of plugins.

v1.3 – November 27, 2011 – Fixed an issue that prevented the jQuery form validation plugin to load and the contact form to be submitted.

v1.2 – May 13, 2011 – Template files were updated to include:

  • FancyBox plugin generated slow loading times or even crashed in IE7 and IE8 browsers. We have replaced FancyBox with ColorBox and the issue has been solved;
  • Minor modification to the CSS file for a better spacing of the “Contact” title of the contact form.
v1.1 – April 11, 2011 – Files have been updated to include the following bug fixes:
  • Fixed the “Print” button action;
  • Adjusted the script to allow the use of ‘light’ and ‘dark’ color schemes;
